Hit endpoint

Daily-unique by IP + UA + day:

GET /api/hit/<key>

Example: /api/hit/profile

Badge endpoints

GET /api/svg/<key>
GET /api/badge/<key>

Try: /api/svg/profile

Pixel (optional)

1×1 transparent PNG with no-cache headers:

GET /api/pixel/<key>

/api/pixel/profile

Admin

Reset (requires x-admin-token):

POST /api/admin/reset { key }

Inspect today visitors (requires x-admin-token):

GET /api/admin/visitors/<key>
